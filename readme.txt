A simple plugin to use etiquetags editor in wordpress blogs

== Description ==

This plugin helps you to use **etiquetags** editor in your WordPress Blog.

* etiquetags is a full-featured, clean, fast and easy-to-use HTML tags editor.

* With etiquetags you will be able to **create automatically lists, tables, link, objects** ...

* Lots of features are included in this new version: the **quotations** tool, the **frequent classes**  menu, the **CSS** button ... and there are others already present in the first version: **footnotes** tool, fast link tool and more ...

== Installation ==

1. Upload `myetiquetags` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
